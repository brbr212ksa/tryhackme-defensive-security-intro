## Description
My second TryHackMe room. A beginner-friendly introduction to defensive security, covering the basics of Blue Teaming, Threat Intelligence, and Security Operations Center (SOC) fundamentals.

## Overview
This room introduces the core concepts of defensive security. It explains how defenders protect organizations, detect cyber attacks, and respond to incidents. It covers key areas like SIEM (Security Information and Event Management), Incident Response (IR), and Malware Analysis.

## Objectives
* Understand the role of Defensive Security (Blue Team).
* Learn about Security Operations Centers (SOC) and Threat Intelligence.
* Explore SIEM tools and how they are used in logging and monitoring.
* Perform basic log analysis to detect malicious activity.

## Tools & Skills Discussed
* Log Analysis concepts
* SIEM (Security Information and Event Management)
* Incident Response fundamentals

## Steps
1. **Understanding Blue Teaming:** Learned the difference between offensive and defensive security.
2. **Exploring SOC:** Understood how a Security Operations Center monitors and responds to threats.
3. **Log Analysis:** Investigated simulated logs using SIEM concepts to identify a cyber attack (e.g., finding malicious IPs or failed login attempts).
4. **Incident Response:** Learned the basic steps of responding to a security breach.

## Key Concepts
* Defensive Security (Blue Team)
* SOC (Security Operations Center)
* SIEM (Security Information and Event Management)
* Malware Analysis
* Cyber Threat Intelligence (CTI)

## What I Learned
* The importance of monitoring and log analysis in detecting attacks using SIEM.
* How to analyze security events to trace back the steps of an attacker.
* The structured approach to handling security incidents.
